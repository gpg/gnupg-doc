The GnuPG Repositories

Use our GIT Viewer to browse the GIT repository. Activity reports are also available.

Here is a list of shortcuts to often used repositories:

  • GnuPG
    • View the GnuPG development branch
    • View the GnuPG stable (2.2) branch
    • View the GPGME (GnuPG Made Easy) library
  • Libgcrypt
    • View the Libgcrypt development branch
  • Gpg4win
    • View the Gpg4win installer
    • View GpgOL (Outlook plugin)

For actual work you should clone a repository; use 

  git clone https://dev.gnupg.org/source/foo.git

or

  git clone git://git.gnupg.org/foo.git

and replace foo by the name of the project (e.g. gnupg or libgcrypt). Many commits and all tags are signed using GnuPG. To verify the signatures on a tag (e.g. gnupg-2.2.20) do 

  git tag -v gnupg-2.2.20
